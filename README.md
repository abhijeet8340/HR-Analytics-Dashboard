## 📌 Project Overview
This project is an **HR Analytics Dashboard** built using Power BI to analyze employee data and uncover key insights related to **attrition, salary, age distribution, job roles, and employee demographics**.

The dashboard helps HR teams make **data-driven decisions** to improve employee retention and workforce planning.

---

## 🎯 Key Objectives
- Analyze employee attrition trends
- Identify factors affecting employee turnover
- Monitor workforce demographics
- Provide actionable insights for HR decision-making

---

## 📊 Dashboard Features

### 🔹 Key Metrics (KPIs)
- Total Employees: **1.47K**
- Attrition Count: **237**
- Attrition Rate: **16.1%**

## 📈 Business Insights
- Highest attrition observed in **26–35 age group**
- Employees with **lower salary slabs (≤5K)** have higher attrition
- **Laboratory Technicians & Sales Executives** show higher attrition rates
- Job satisfaction impacts employee retention significantly
